|---------------- | --------------- | --------------- |
| change    | c     | 修改    |

| ~ , g~  |      | 大小写转换   |
| gu  |      | 转成小写   |
| gU  |      | 转换成大写   |

# Positing
## Phyiscal
| 命令    | 命令缩写    | 功能    |
|---------------- | --------------- | --------------- |
| :n     |       | 跳转到第n行    |
| gg, C-Home    |   |  跳转到第1行     |
| G, C-End   |      | 跳转到最后一行   |
| n%   |      | 跳转到%n行（百分号）   |
| +,j |  | 向下移动1行 |
| -,k |  | 向上移动1行 |
